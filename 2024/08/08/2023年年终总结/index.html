会想你的 我四年鲜活的青春

对这所城市有复杂的感情，苦甜参半。
刚入学时，经常被厦门的天空浪漫到，手机里街景几百张几百张地增加。记得刚来的时候，踩着学校的上限报名了学生部门和社团，每天都好忙好忙，没有时间管学习，各种部门的团建，破冰，或者部门任务。大一的经院好声音天天排练到凌晨，那时候还会和好友一起半夜大排档K歌，然后直接去海边看日出，好浪漫好浪漫。还有每周日一次的数分小测，从开始什么都听不懂到后来慢慢的有点乐在其中，现在回忆起来，数分还是我最喜欢的课。
四年过去，慢慢地也对海岛，椰树，日落脱敏了。认识的人真的很多，但好像也没交到很好的朋友。想起zqjj说的：“我在大一一开始的时候能量开太大了，后面就完全不想社交了”。我没那么夸张但也差不多吧。

临别的时候有了毕业的实感，但说实话我并不很伤感，离开厦门的时候甚至更多的是解放的快感。以后肯定会怀念，但此刻我真的没什么眷恋。

话虽这么说，仪式感是要有的，和一些重要的朋友都留下了合影，早些时候和xyjj约好毕业季要去翔安框框一顿拍，但实际上到了最后不然她没空不然我没空，也许工作之后就更是如此了吧，异地之后，一年见一次都难。或许也是从一开始就知道，大学里的朋友都五湖四海，毕业之后如果不在一个城市，可不是相忘于江湖嘛。也正是抱着这样的想法，大学毕业的时候，反而没有高中毕业伤感。有的人毕业那一天，你就知道是最后一面。2024年的五月底开始骶髂关节疼，一直到八月还没完全康复（但快好了！fighting——这里插播提醒一下自己，真的要开始加强锻炼了！）就更加没有了拍照的心情。和西部梦想贵州队拍的那个早上算是最开心的了哈哈哈哈，不过其实也是强忍着疼去拍的。不过西梦算是我大学最喜欢的部门了，比文中什么的好多了（文中什么东西啊*？学生官僚让你们玩明白了笑死）

忏悔：这一年游戏打少了

今年感受到了一些心态上的改变，觉得自己浮躁很多, 从打游戏上就能看出来。虽然这个是从一开始很早就发现了的，但是今年显得尤为明显。

四年前的我：肉鸽游戏如果失败了还随便可以从当前层重开就完全没有意思
现在的我：这游戏打到第七层了死了还要从头再来？也太打击了吧

1.失败了几十次、几百次之后技术就会显著提高。并且会越打越难。
2.会逐渐攻克解锁所有的道具和开局。
3.会因为偶尔天胡组合感到兴奋。
4.其实肉鸽和人生一样毫无意义，但是喜欢玩就是意义，喜欢的人能看见各种各样的玩法，玩得有趣

和朋友聊天的时候我说有一类人是注定厉害。这种人纯纯的 learn for nothing, learn for fun.
他们和那种learn for profit的shit完全不一样，我是打心眼里佩服。
某些瞬间我也觉得自己变成了这种我佩服的类型，但对现在的我来讲，当初那种learn for nothing, learn for fun的感觉退化了很多（难道我也变成shit了吗（怒）

（btw后来没多久被绿了因为茶小姐看上了一班的帅哥）
不过世界真的很小，后来高二分班之后的同桌是范景涵的姐姐，于是跟着又听了一堆黑料。貌似我发小也喜欢过这个男生，我发小也是智性恋我猜也是因为fjh数学好，大家都叫他范教授，但是其实现在感觉也没特别厉害我感觉我对象更强一点哈哈哈哈哈哈哈，甚至高考数学还没我高呢，但是人家确实是失误了。后来从我闺蜜嘴里听说他初中喜欢外国语的szy，szy也是我朋友啊这个世界真的要这么小吗，不过szy是lesbian啊你知道吗范教授😂

王家东是因为长得帅，高高帅帅喜欢打篮球，那时候和白岚臻每周都在期待周五的体育课，把热身时间溜了去偷看33班男生打球。她看王昊辰，我看王家东。
怎么认识的呢？初一一起检查眼保健操，我们俩都不去好好查，溜到27班的阅览室和左安聊天（左安是27班另一个同学，反面教材学生模范，后来教导主任说左安是个混子流氓，再也不准我们玩了）。
那时候还有很烂俗的桥段，放学和白岚臻特地绕路去制造偶遇，就从教学楼走到校门口这段五分钟的路，聊个三五句话都觉得很开心。其实也有朋友加成，王家东和王昊辰恰好是基友，天天同出同进，我和白岚臻也恰好关系好，所以那段时间天天黏在一起偷偷看喜欢的男生。我甚至还记得那是周五第七节课，距离周末不到一小时，本来就是一周中最快乐的时光，还能看见喜欢的人，我和白岚臻偷偷打赌，他们俩谁能进的球更多，事实证明王家东打球是很厉害的o(￣▽￣)ブ。黄昏的时候篮球场上进球的白衣少年，真的和言情小说里写的一样啊……
我们认识了不到三年，这段感情也随着毕业无疾而终。三年里始终保持着每天能见个面聊天的关系，我便也认识了他的小弟李成昊（一个常年高原红的搞笑男），我记得王家东会欠欠地和人打招呼，人会频繁地喜欢上同一类人是真的，我都喜欢会犯贱的类型。
再后来，省中的自主招生比中考提前了一个月，我成功避开了那段最紧张焦虑的日子提前进入高中，中考复习期间他不怎么用QQ，我们也便断了联系。因为知道他成绩很差，后来我甚至不敢打听他去了哪所高中，我甚至不知道他去了职高还是高中。但抓马的是他小弟李成昊成绩很好，也去了教改1班，就在我隔壁，高中开学了两个月之后我假装随口问了一下诶王家东去哪啦？李成昊却说他也不知道。
这就是结局了，现在回忆起来还是很美好。暗恋真的很奇怪啊！有的时候你完全不在意结果，也不在乎他喜不喜欢你，单纯是因为学校有一个你喜欢的人就会觉得每一天都有盼头。

暗恋这种拧巴的感觉，你也一定懂吧。
